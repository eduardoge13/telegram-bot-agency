    ```bash
    cd /opt

    ```

    **Step 3 — Configure nginx reverse proxy:**
    Create `/etc/nginx/sites-available/puntoclavemx`:
    ```nginx
    server {
        listen 80;
        server_name _;
        location / {
            proxy_pass http://localhost:3000;
            proxy_http_version 1.1;
            proxy_set_header Upgrade $http_upgrade;
            proxy_set_header Connection 'upgrade';
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto $scheme;
            proxy_cache_bypass $http_upgrade;
        }
    }
    ```
    Enable it:
    ```bash
    sudo ln -s /etc/nginx/sites-available/puntoclavemx /etc/nginx/sites-enabled/
    sudo nginx -t && sudo systemctl reload nginx
    ```
    NOTE: If nginx already has a default site on port 80, remove `/etc/nginx/sites-enabled/default` or use a domain-based `server_name`.
